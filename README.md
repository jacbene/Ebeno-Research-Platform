# Plateforme de Recherche Ebeno

Plateforme de recherche collaborative conçue pour les Sciences Humaines et Sociales (SHS). Ce projet est une application web monorepo qui offre une suite d'outils complète pour assister les chercheurs dans leurs projets d'analyse qualitative.

## Fonctionnalités Principales

- **Gestion de Projets :** Créez et organisez vos projets de recherche.
- **Transcription :** Téléchargez et transcrivez des fichiers audio et vidéo.
- **Analyse Qualitative :** Codez et annotez vos documents et transcriptions.
- **Gestion des Références :** Organisez vos sources bibliographiques.
- **Création d'Enquêtes :** Concevez et déployez des enquêtes pour votre collecte de données.
- **Visualisations de Données :** Analysez vos données avec des graphiques interactifs (nuages de mots, fréquences de codes, etc.).
- **Collaboration :** Travaillez en équipe sur les mêmes documents et projets.
- **Assistance IA :** Utilisez l'intégration DeepSeek pour vous aider dans l'analyse.

## Structure du Projet

Le projet est organisé en monorepo et contient les packages principaux suivants :

- `frontend` : Une application cliente riche construite avec React, TypeScript et Vite.
- `backend` : Une API serveur robuste construite avec Node.js, Express et Prisma pour la gestion de la base de données.
